# YKO Portfolio

A terminal-style portfolio website built with Next.js, React, and TailwindCSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - React components
  - `panels/` - Display panel components (Welcome, Whoami, Skills, etc.)
- `app/globals.css` - Global styles and TailwindCSS configuration

## Features

- Terminal-style interface with command history
- Interactive command system
- Responsive design (mobile-friendly)
- CRT scanline effect
- Smooth animations and transitions

## Commands

Type `help` in the terminal to see all available commands.

## Build

To build for production:

```bash
npm run build
npm start
```
